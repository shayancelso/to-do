# To-Do List

Personal task management with Claude Code integration.

## Usage

In Claude Code, use the `/do` command:

```
/do buy groceries tomorrow priority:high #shopping
/do list
/do done 1
/do delete 2
```

## Files

- `tasks/tasks.md` - Human-readable task list (view on phone)
- `tasks/tasks.json` - Machine-readable for integrations
